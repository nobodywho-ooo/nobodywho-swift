# NobodyWho for Swift

On-device AI inference for Swift apps. Run LLMs locally on iOS, macOS, visionOS, and watchOS with no API keys or network connection required.

NobodyWho is powered by [llama.cpp](https://github.com/ggml-org/llama.cpp) and supports thousands of GGUF models from [Hugging Face](https://huggingface.co/models?library=gguf&sort=trending).

You can see and contribute to NobodyWho inference engine code [here](https://github.com/nobodywho-ooo/nobodywho).

## Installation

Add NobodyWho to your project using Swift Package Manager:

```swift
dependencies: [
    .package(url: "https://github.com/nobodywho-ooo/nobodywho-swift.git", from: "1.0.0"),
]
```

Or in Xcode: **File > Add Package Dependencies** and enter `https://github.com/nobodywho-ooo/nobodywho-swift`.

### Platform Support

| Platform | Minimum Version |
|----------|----------------|
| iOS      | 15.0           |
| macOS    | 13.0           |
| visionOS | 1.0            |
| watchOS  | 10.0           |

GPU acceleration is available via Metal on iOS, macOS, and visionOS. watchOS runs on CPU only.

## Quick Start

```swift
import NobodyWho

let chat = try await Chat.fromPath(
    modelPath: "hf://NobodyWho/Qwen_Qwen3-0.6B-GGUF/Qwen_Qwen3-0.6B-Q4_K_M.gguf",
    systemPrompt: "You are a helpful assistant."
)

// Stream tokens as they are generated
for await token in chat.ask("What is the capital of Denmark?") {
    print(token, terminator: "")
}

// Or get the complete response
let response = try await chat.ask("What is the capital of Denmark?").completed()
```

## Usage

### Loading Models

Models can be loaded from a local file path, a Hugging Face repository, or any HTTPS URL:

```swift
// From Hugging Face
let model = try await Model.load(modelPath: "hf://NobodyWho/Qwen_Qwen3-0.6B-GGUF/Qwen_Qwen3-0.6B-Q4_K_M.gguf")

// From a local file
let model = try await Model.load(modelPath: "/path/to/model.gguf")

// With download progress
let model = try await Model.load(modelPath: "hf://...") { downloaded, total in
    print("Downloaded \(downloaded)/\(total) bytes")
}
```

A single `Model` can be shared across multiple `Chat`, `Encoder`, or `CrossEncoder` instances:

```swift
let model = try await Model.load(modelPath: "hf://...")
let chat1 = Chat(model: model, systemPrompt: "You are a pirate.")
let chat2 = Chat(model: model, systemPrompt: "You are a poet.")
```

### Tool Calling

Define tools that the model can invoke during inference. The `@DeclareTool` macro generates the boilerplate from your function signature:

```swift
import NobodyWho

@DeclareTool("Get the current weather for a city")
func getWeather(city: String, unit: String) -> String {
    return "{\"temp\": 22, \"unit\": \"\(unit)\"}"
}

let chat = try await Chat.fromPath(
    modelPath: "hf://...",
    systemPrompt: "You are a helpful assistant.",
    tools: [getWeatherTool]  // generated by @DeclareTool
)

let response = try await chat.ask("What's the weather in Paris?").completed()
```

For tools defined inside function bodies (where peer macros can't be used), use the initializer directly:

```swift
let weatherTool = Tool(
    name: "get_weather",
    description: "Get the current weather for a city",
    parameters: [("city", #"{"type": "string"}"#), ("unit", #"{"type": "string"}"#)]
) { args in
    let city = args[0] as! String
    let unit = args[1] as! String
    return "{\"temp\": 22, \"unit\": \"\(unit)\"}"
}
```

### Multimodal Input

Send images and audio alongside text using vision/audio models:

```swift
let model = try await Model.load(
    modelPath: "path/to/vision-model.gguf",
    projectionModelPath: "path/to/mmproj.gguf"
)
let chat = Chat(model: model, systemPrompt: "Describe what you see.")

let prompt = Prompt([
    Prompt.image("photo.png"),
    Prompt.text("What is in this image?"),
])
let response = try await chat.ask(prompt).completed()
```

### Embeddings

Generate text embeddings for semantic search and similarity:

```swift
let encoder = try await Encoder.fromPath(modelPath: "embedding-model.gguf")
let embedding = try await encoder.encode("Hello world")

let a = try await encoder.encode("cat")
let b = try await encoder.encode("kitten")
let similarity = cosineSimilarity(a: a, b: b)
```

### Cross-Encoder Reranking

Rank documents by relevance to a query:

```swift
let crossEncoder = try await CrossEncoder.fromPath(modelPath: "reranker.gguf")
let ranked = try await crossEncoder.rankAndSort(
    query: "How to reset password?",
    documents: ["Reset your password in settings.", "Our menu has pizza.", "Contact support for help."]
)
// ranked: [("Reset your password in settings.", 0.95), ("Contact support for help.", 0.42), ...]
```

### Sampler Configuration

Control text generation behavior with sampler presets:

```swift
let chat = try await Chat.fromPath(
    modelPath: "hf://...",
    sampler: SamplerPresets.temperature(0.7)
)

// Available presets:
// SamplerPresets.default()
// SamplerPresets.greedy()
// SamplerPresets.temperature(0.7)
// SamplerPresets.topK(40)
// SamplerPresets.topP(0.9)
//
// Structured output constraints (via llguidance):
// SamplerPresets.constrainWithJsonSchema("{...}")  — constrain to a JSON Schema
// SamplerPresets.constrainWithRegex("[0-9]+")      — constrain to a regex pattern
// SamplerPresets.constrainWithGrammar("...")        — constrain to a Lark or GBNF grammar
```
